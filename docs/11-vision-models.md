# 11. Запуск vision-моделей (Vision tab)

Вкладка **Studio → Vision** прогоняет модели зоопарка на NPU по реальной
картинке: фото из галереи или снимок с камеры.

## Что поддержано

| Категория | Что показывает | Как считается |
|---|---|---|
| Classification | топ-5 классов ImageNet с процентами | softmax → top-5, метки из `labels.txt` модели или из встроенного `assets/vision/imagenet_classes.txt` |
| Depth | цветная карта глубины | min-max нормализация → turbo-colormap |
| Segmentation | оверлей классов | argmax по канальной оси (NCHW/NHWC) → палитра |
| Super-resolution | увеличенное изображение | рендер выходного RGB-тензора `[0,1]` |
| Detection / Pose / прочее | пока только время + статистика выхода | рендер требует модель-специфичного декодинга (anchors/NMS) |

## Как работает препроцессинг

- геометрия входа (NCHW `[1,3,H,W]` vs NHWC `[1,H,W,3]`, число каналов и
  размер) **читается из самой модели** — `InputGeometry.parse(model.inputs[0].shape)`;
- картинка ресайзится под `H×W` и подаётся как RGB в диапазоне `[0,1]` — это
  конвенция AI Hub (ImageNet-нормализация зашита в граф). Если вход целочисленный
  (`uint8`), значения масштабируются в `[0,255]`;
- если классификатор почему-то выдаёт неправильные метки — почти наверняка
  конкретная модель ждёт иную нормализацию входа; это единственное место, где
  без устройства нельзя было выверить все 12 классификаторов одинаково.

## Камера и доступ к файлам — без лишних разрешений

- **Камера**: `ActivityResultContracts.TakePicturePreview()` — снимок делает
  системное приложение камеры, поэтому **разрешение `CAMERA` приложению не
  нужно** (и оно намеренно не объявлено).
- **Галерея** и **импорт своей модели**: SAF `GetContent` — Storage Access
  Framework тоже не требует runtime-разрешений.

## Свои модели без adb

**Models → Import from storage** принимает `.bin`, `.dlc` или `.zip`
(в т.ч. немодифицированный архив релиза AI Hub). Файлы кладутся в
`models/custom/` и автоматически появляются в Benchmark и (если это
картиночная модель) в Vision. См. также [10-custom-models.md](10-custom-models.md).
